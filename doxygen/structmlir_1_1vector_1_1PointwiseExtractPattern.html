Detailed Description

Canonicalize an extra element using the result of a pointwise operation.

Transforms: v = addf a, b : vector32xf32> dv = vector.extract_map v, id, 32 : vector<32xf32> into vector<1xf32> to: da = vector.extract_map a, id, 32 : vector<32xf32> into vector<1xf32> db = vector.extract_map a, id, 32 : vector<32xf32> into vector<1xf32> dv = addf da, db : vector<1xf32>
