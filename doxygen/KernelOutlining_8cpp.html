◆ extractBeneficiaryOps()

static bool extractBeneficiaryOps ( Operation *  op,
llvm::SetVector< Operation *> &  beneficiaryOps,
llvm::SetVector< Value > &  availableValues 
)
static

For a given operation op, computes whether it is beneficial to sink the operation into the kernel.

An operation can be sunk if doing so does not introduce new kernel arguments. Whether a value is already available in the kernel (and hence does not introduce new arguments) is checked by querying availableValues. If an operand is not yet available, we recursively check whether it can be made available by siking its defining op. Operations that are indentified for sinking are added to beneficiaryOps in the order the should appear in the kernel. Furthermore, availableValues is updated with results that will be available after sinking the identified ops.
